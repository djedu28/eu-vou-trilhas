## Para usar basta rodar

* `yarn` para baixar os pacotes
* `yarn dev`  para executar o projeto em modo desenvolvimento

se não possuir o `yarn` instalado, pode usar o `npm` ou o seu gerenciador de pacotes preferido

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
